# Sprettur Website

## Requirements

* [NodeJS 0.8](http://nodejs.org/)
* [Grunt 0.4](http://gruntjs.com/)

## Building

    $ npm install
    $ bower install
    $ grunt server


## Deploying

    $ grund build
